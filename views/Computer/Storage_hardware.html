# SSD

颗粒
简称 全称 规格
SLC Single-Level Cell 1 bit per cell
MLC Multi-Level Cell 2 bit per cell
TLC Triple-Level Cell 3 bit per cell
QLC Quad-Level Cell 4 bit per cell

  • 10000 RPM

  • SATA

版本 理论速度

版本 带宽 Column 0 Column 1 Column 2 Column 3 Note
